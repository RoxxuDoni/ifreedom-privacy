Политика конфиденциальности I-Fredom

Дата обновления: 30 января 2026

1. Кто мы

I-Fredom (“мы”, “приложение”) — мобильное приложение для Android, предоставляющее VPN-соединение через механизм Android VpnService.

2. Активационный ключ

Для доступа к функциональности приложения может потребоваться активационный ключ. Ключ предоставляется пользователю отдельно. Оплата и платёжные данные в приложении не обрабатываются.

3. Какие данные мы собираем

Мы не собираем и не продаём персональные данные пользователей. Приложение может обрабатывать локально на устройстве данные, необходимые для работы VPN, например:

  • выбранный сервер и технические параметры подключения;
  • статус VPN (включено/выключено);
  • технические показатели качества соединения (например, пинг).

4. Что происходит с интернет-трафиком

При включении VPN сетевой трафик устройства направляется через VPN-туннель к выбранному серверу. Мы не используем VPN для подмены рекламы, скрытого перенаправления трафика или продажи данных.

5. Передача данных третьим лицам

Мы не передаём персональные данные третьим лицам, кроме случаев, предусмотренных законом.

6. Хранение и безопасность

Мы применяем разумные меры для защиты данных, которые приложение хранит локально. Пользователь может удалить данные приложения стандартными средствами Android (Настройки → Приложения → I-Fredom → Память → Очистить).

7. Дети

Приложение не предназначено для детей младше 13 лет и не собирает сознательно данные детей.

8. Изменения политики

Мы можем обновлять эту политику. Актуальная версия всегда доступна на этой странице.
